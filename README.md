# StudGen
Генератор отчетов об успеваемости студентов. Программа обрабатывает список студентов с оценками и посещаемостью и выдает отчет, включающий в себя средний балл для каждого студента, определение студентов с самой высокой и низкой успеваемостью и посещаемостью, информацию о наиболее и наименее посещаемых занятиях
